Aperçu concret

Ce que vous manipulerez

De la carte thématique construite au clic dans QGIS… au script Python qui la reproduit automatiquement.

Console Python — QGIS
# Reprojeter 50 couches d'un coup
import processing, glob

for shp in glob.glob("donnees/*.shp"):
    processing.run("native:reprojectlayer", {
        "INPUT": shp,
        "TARGET_CRS": "EPSG:32631",
        "OUTPUT": shp.replace(".shp",
                           "_utm.shp")
    })
print("50 couches reprojetées !")
